# Data Engineering Agent

&lt;!— ESTE ES UN BORRADOR —>

Data Engineering Agent es un copiloto basado en IA y con tecnología de Agent Orchestrator para equipos de datos en Adobe Experience Platform. El agente está diseñado para gestionar el trabajo pesado en todo el ciclo de vida de los datos de extremo a extremo: modelado de datos, incorporación, preparación de datos SQL, gobernanza y administración del ciclo de vida. Con Data Engineering Agent, los ingenieros y arquitectos pueden moverse más rápido con una mayor calidad de datos y menos trabajo manual.

Data Engineering Agent está organizado en habilidades que puede aprovechar para optimizar los flujos de trabajo.

| aptitud de Data Engineering Agent | Descripción |
| --- | --- |
| Incorporación de datos | <strong>La incorporación de datos</strong> admite orígenes por lotes como S3, Marketo y Data Landing Zone (DLZ). Entre sus capacidades se incluyen: <ul><li>Conexión a varias fuentes de datos, incluidas S3, Data Landing Zone, Marketo y más.</li><li>Datos de origen de generación de perfiles (por ejemplo, valores distintos, valores nulos, duplicados y métricas de calidad básicas).</li><li>Alinear campos de origen tanto a grupos de campos XDM estándar como a campos personalizados.</li><li>Proponer y crear flujos de datos de ingesta de datos (por ejemplo, mover datos de S3 o Marketo a conjuntos de datos de RTCDP/CJA).</li></ul> |
| Calidad de los datos automatizada y enriquecimiento semántico | <ul><li>Realiza evaluaciones exhaustivas de la calidad de los datos, incluida la identificación de anomalías y la notificación de recuentos de registros válidos o no válidos.</li><li>Recomienda mejoras en la calidad de los datos, como normalización de tipos, formato de moneda y deduplicación de ID.</li><li>Aplica enriquecimiento semántico a esquemas basados en muestreo de datos y reconocimiento inteligente de patrones de nomenclatura.</li><li>Involucra a los usuarios para que confirmen los enriquecimientos sugeridos, lo que garantiza cambios de datos finalizados y aprobados.</li></ul> |

Con Data Engineering Agent, puede asegurarse de lo siguiente:

- **Incorporación más rápida**: reduzca el tiempo de semanas de asignación manual de esquemas y configuración de canalizaciones a horas de configuración conversacional y guiada.
- **Mayor calidad de datos**: menos registros no válidos e incidentes de producción debido al análisis de calidad de datos integrado y a las comprobaciones semánticas.
- **Gobernanza y cumplimiento mejorados**: Las directivas de gobernanza se adjuntan en tiempo de diseño (etiquetas, TTL, control de identidad) en lugar de considerarse como una idea tardía.
- **Equipos de datos más productivos**: el trabajo repetitivo de SQL/flujo de datos está automatizado para que los ingenieros se centren en el diseño y la optimización de mayor valor.
- **Autoservicio más amplio**: las partes interesadas no expertas pueden autoabastecer con seguridad las tareas de preparación de datos comunes con protecciones.
